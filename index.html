Práctica N° 3

1. Crear un programa para solicitar un numero n para luego generar una tabla n x n que contenga en sus celdas números desde 1 hasta n x n.

2. Crear un programa donde se defina un arreglo para almacenar la cantidad de habitantes que tiene los departamentos de Bolivia, luego desplegarlos en una tabla con un fila que muestre la suma total.

3. Crear un programa para averiguar si un número es perfecto. (Un número perfecto es un número que es igual a la suma de sus divisores propios positivos. Ej. 28 = 1 + 2 + 4 + 7 + 14)

4. Crear un programa en PHP que reciba una cadena que contiene una fecha en formato YYYY-mm-dd y lo convierta en formato dd-mm-YYYY.

5. Crear un programa que muestre las imágenes que contiene un directorio en cuatro columnas, además al pulsar sobre cualquier imagen debe mostrar la imagen en tamaño real (En la carpeta puede existir cualquier número de imágenes)

6. Realizar un programa que reciba una fecha en formato numérico y lo convierta a texto. (Ejemplo: 21/09/2017 => 21 de septiembre de 2017)

7. Desarrollar un programa para convertir un número a texto. Ejemplo: entrada 125, salida: ciento veinte cinco.

8. Crear un programa para simular el lanzamiento de dos (2) dados. (sugerencia: utilizar la función rand())

9. Crear un programa que calcule el salario semanal de un obrero que trabaja 8 horas al día. La variable de entrada es el costo por hora que se le paga.

10. Desarrollar un programa que guarde 10 preguntas sobre un tema y muestre aleatoriamente tres preguntas.
